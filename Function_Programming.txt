Lambda function:(also called anonymous function)
() -> {}
() => parameter
{} => body
--it always works with functional interface(interface with only one method)

Built-in Functional Interface
1.Predicate: an interface that takes something and returns true or false
                Predicate<T> => T:object datatype

2.Consumer: an interface which takes input and does something with it and returns nothing
            input->action->return nothing: print number, updating value


3.Supplier:
--interface does not take any input and produces a value => using get() value
Ex: Random number, create new object
